5变量本质：内存中的标签

Python 中的变量不是"盒子"，而是贴在内存对象上的标签。多个变量可以指向同一个对象。
变量赋值本质
a = [1, 2, 3]
b = a        # b 和 a 指向同一个列表对象！
b.append(4)
print(a)     # [1, 2, 3, 4] — a 也变了！

# 想要复制？用 copy()
c = a.copy() # c 是新对象
c.append(5)
print(a)     # [1, 2, 3, 4] — a 不变
is vs ==：==比较值是否相等；is比较是否是同一个对象（内存地址相同）。小整数（-5~256）和短字符串会被缓存，所以 a is b 可能为 True。

6字符串切片完全指南

关键规则：if/elif/else 后面必须有冒号 :；条件为真时执行缩进的代码块；Python用缩进表示代码层次，不是大括号！

2紧凑形式（条件表达式）

一行写if-else
# 紧凑形式：结果A if 条件 else 结果B
grade = "及格" if score >= 60 else "不及格"

# 等价于
if score >= 60:
    grade = "及格"
else:
    grade = "不及格"
考试常考：紧凑形式是 Python 特有的语法，注意顺序是 A if 条件 else B，不是 if 条件 then A else B。

def bubble_sort(arr):
    n = len(arr)
    for i in range(n-1):
        swapped = False
        for j in range(n-1-i):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr

线性回归基本要素

线性回归是最基础的监督学习模型，目标是学习输入特征与输出之间的线性映射关系。
模型定义
ŷ = w1x1 + w2x2 + ... + b，w为权重，b为偏置
损失函数
均方误差 MSE = (1/n) * Sum((y_hat - y)^2)
优化算法
小批量随机梯度下降 Mini-batch SGD，沿梯度反方向更新参数
线性回归三要素
import torch

x = torch.randn(100, 1)

ResNet残差网络

残差连接通过跨层连接解决深层网络的梯度消失和退化问题，使得训练数百层的网络成为可能。
残差块实现
class ResidualBlock(nn.Module):
    def __init__(self, channels):
        super().__init__()
        self.conv1 = nn.Conv2d(channels, channels, 3, padding=1)
        self.bn1 = nn.BatchNorm2d(channels)
        self.conv2 = nn.Conv2d(channels, channels, 3, padding=1)
        self.bn2 = nn.BatchNorm2d(channels)
    def forward(self, x):
        residual = x  # 保存恒等映射
        out = torch.relu(self.bn1(self.conv1(x)))
        out = self.bn2(self.conv2(out))
        out += residual  # 跨层连接（skip connection）
        return torch.relu(out)

残差块学习的是 F(x) = H(x) - x，网络只需学习残差映射，降低了学习难度。

5卷积核尺寸计算公式

输出特征图尺寸 = floor((输入尺寸 + 2×padding - kernel_size) / stride + 1)
计算示例

RNN基本原理

循环神经网络(RNN)通过隐状态在时间步之间传递信息，适合处理序列数据。
隐状态 h_t
在时间步 t 融合当前输入 x_t 和上一时刻隐状态 h_{t-1}
时间步
序列中每个位置的计算单元，依次展开处理
梯度问题
长序列训练时出现梯度消失或梯度爆炸
RNN计算公式
# 隐状态更新: h_t = tanh(W_hh * h_{t-1} + W_xh * x_t + b_h)
# 输出:       y_t = W_hy * h_t + b_y

import torch.nn as nn
rnn = nn.RNN(input_size=10, hidden_size=20, batch_first=True)
x = torch.randn(32, 5, 10)  # batch=32, seq_len=5, input=10
output, h_n = rnn(x)
print(output.shape)  # [32, 5, 20]
print(h_n.shape)     # [1, 32, 20] - 最后时刻隐状态

LSTM长短期记忆

LSTM 通过门控机制（输入门、遗忘门、输出门）解决 RNN 的长程依赖和梯度消失问题。
门作用公式
遗忘门 f_t决定丢弃哪些旧记忆sigma(W_f * [h_{t-1}, x_t] + b_f)
输入门 i_t决定存入哪些新信息sigma(W_i * [h_{t-1}, x_t] + b_i)
候选记忆 c~_t生成新的候选值tanh(W_c * [h_{t-1}, x_t] + b_c)
输出门 o_t决定输出哪些信息sigma(W_o * [h_{t-1}, x_t] + b_o)

lstm = nn.LSTM(input_size=10, hidden_size=20, num_layers=2,

控制旧信息的忽略程度

def compute_iou(box1, box2):
    # box格式: [x1, y1, x2, y2]
    x1 = max(box1[0], box2[0])
    y1 = max(box1[1], box2[1])
    x2 = min(box1[2], box2[2])
    y2 = min(box1[3], box2[3])
    inter = max(0, x2 - x1) * max(0, y2 - y1)
    area1 = (box1[2]-box1[0]) * (box1[3]-box1[1])
    area2 = (box2[2]-box2[0]) * (box2[3]-box2[1])
    return inter / (area1 + area2 - inter + 1e-6)
NMS非极大值抑制
def nms(boxes, scores, threshold=0.5):
    order = scores.argsort(descending=True)
    keep = []
    while order.numel() > 0:
        keep.append(order[0].item())
        if order.numel() == 1: break
        ious = [compute_iou(boxes[order[0]], boxes[j])
                 for j in order[1:]]
        mask = [iou < threshold for iou in ious]
        order = order[1:][torch.tensor(mask)]
    return keep

TextCNN模型
class TextCNN(nn.Module):
    def __init__(self, vocab_size, embed_dim, num_classes):
        super().__init__()
        self.embed = nn.Embedding(vocab_size, embed_dim)
        self.convs = nn.ModuleList([
            nn.Conv1d(embed_dim, 128, k)  # k=3,4,5 多尺度
            for k in [3, 4, 5]
        ])
        self.dropout = nn.Dropout(0.5)
        self.fc = nn.Linear(128 * 3, num_classes)
    def forward(self, x):  # x: [batch, seq_len]
        x = self.embed(x).permute(0, 2, 1)  # [B, E, L]
        out = [torch.relu(conv(x)).max(dim=2)[0]
               for conv in self.convs]
        out = torch.cat(out, dim=1)
        return self.fc(self.dropout(out))

Conv1d 在嵌入维度上做卷积，不同 kernel_size 捕获不同长度的 n-gram 特征。

class Encoder(nn.Module):
    def __init__(self, vocab_size, embed_dim, hidden_dim):
        super().__init__()
        self.embed = nn.Embedding(vocab_size, embed_dim)
        self.rnn = nn.GRU(embed_dim, hidden_dim, batch_first=True)
    def forward(self, x):
        return self.rnn(self.embed(x))[1]  # 返回h_n

    def __init__(self, vocab_size, embed_dim, hidden_dim):
        super().__init__()
        self.embed = nn.Embedding(vocab_size, embed_dim)
        self.rnn = nn.GRU(embed_dim, hidden_dim, batch_first=True)
        self.fc = nn.Linear(hidden_dim, vocab_size)

注意力机制 Attention

注意力机制让解码器在每一步动态关注输入序列的不同部分，解决了固定长度上下文的瓶颈。
Query(Q)
解码器当前隐状态，表示'我在找什么'
Key(K)
编码器各时间步隐状态，表示'我有什么'
Value(V)
编码器各时间步隐状态的实际内容
注意力权重 alpha
Q与各K的相似度，经softmax归一化
加性注意力实现
def attention(query, keys, values):
    # query: [B, 1, H], keys: [B, T, H], values: [B, T, H]
    scores = torch.bmm(query, keys.transpose(1, 2))  # [B, 1, T]
    alpha = torch.softmax(scores, dim=-1)  # 注意力权重
    context = torch.bmm(alpha, values)    # 加权求和
    return context, alpha

# 在解码器中使用
context, attn_weights = attention(dec_hidden, enc_outputs, enc_outputs)
combined = torch.cat([dec_input_embed, context], dim=-1)

Attention = Q * K^T -> softmax -> alpha * V。可以理解为动态查询数据库（Q在K中搜索，输出V的加权组合）。

4注意力机制可视化理解

注意力机制的核心思想："在翻译/生成每个词时，模型应该关注输入序列中哪些词？"
缩放点积注意力（Scaled Dot-Product Attention）
import math
import torch.nn.functional as F

def attention(Q, K, V, mask=None):
    """Q, K, V: [batch, seq_len, d_model]"""
    d_k = Q.size(-1)
    scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(d_k)
    
    if mask is not None:
        scores = scores.masked_fill(mask == 0, -1e9)
    
    attn_weights = F.softmax(scores, dim=-1)  # 注意力权重
    output = torch.matmul(attn_weights, V)     # 加权求和
    return output, attn_weights

# Q=K=V 时为自注意力（Self-Attention）
Multi-Head Attention：将 Q/K/V 投影到多个子空间，分别计算注意力，再拼接。相当于"多个视角"看同一个序列，捕获不同依赖关系。

数据结构与算法

算法的5个基本特征

  • 可行性：算法中的每一步都能通过基本运算有限次执行实现
  • 确定性：每一步必须有明确的含义，无二义性
  • 有穷性：在有限步后必须终止
  • 有输入：零个或多个输入
  • 有输出：一个或多个输出

复杂度

  • 时间复杂度：算法执行所需基本运算次数，用 O 记号表示
  • 空间复杂度：算法执行所需辅助存储空间
  • 常见时间复杂度：O(1) < O(log n) < O(n) < O(n log n) < O(n^2) < O(n^3)

存储结构

类型线性结构非线性结构
顺序存储数组-
链式存储链表树、图
  • 线性结构：一对一关系（数组、链表、栈、队列）
  • 非线性结构：一对多或多对多（树、图）
栈与队列

栈 (Stack) -- 后进先出 LIFO

  • 只允许在栈顶进行插入和删除操作
  • 操作：push (入栈), pop (出栈)
  • 应用：递归、表达式求值、括号匹配

队列 (Queue) -- 先进先出 FIFO

  • 队尾进、队头出
  • 操作：enqueue (入队), dequeue (出队)
  • 应用：任务调度、BFS遍历

循环队列

  • 队头指针 front，队尾指针 rear
  • 队空条件：front == rear
  • 元素个数公式见 kb4 页
树与二叉树

基本性质

  • 度为0的叶子结点数 n0 = n2 + 1（度为2的结点数加1）
  • 二叉树第 k 层最多有 2^(k-1) 个结点
  • 深度为 k 的二叉树最多有 2^k - 1 个结点
  • 完全二叉树深度为 [log2(n)] + 1

满二叉树 vs 完全二叉树

类型特点
满二叉树每层结点数都达到最大值
完全二叉树除最后一层外每层满，最后一层从左到右连续

遍历方式

  • 前序遍历：根 - 左 - 右 (Preorder)
  • 中序遍历：左 - 根 - 右 (Inorder)
  • 后序遍历：左 - 右 - 根 (Postorder)
  • 层序遍历：按层从左到右
查找技术
查找方法平均比较次数前提条件
顺序查找n / 2无
二分查找log2(n)有序且顺序存储
  • 顺序查找：从头到尾逐个比较，适用于任何线性表
  • 二分查找（折半查找）：每次与中间元素比较，缩小一半范围
  • 二分查找的效率高，但前提是数据必须有序且使用顺序存储
排序对比
排序方法最坏时间复杂度平均时间复杂度
冒泡排序O(n(n-1)/2)O(n^2)
快速排序O(n(n-1)/2)O(n log2 n)
插入排序O(n(n-1)/2)O(n^2)
选择排序O(n(n-1)/2)O(n^2)
堆排序O(n log2 n)O(n log2 n)
归并排序O(n log2 n)O(n log2 n)
  • 冒泡、插入、选择排序的最坏比较次数均为 n(n-1)/2
  • 堆排序和归并排序最坏情况也为 O(n log2 n)
  • 快速排序平均性能最好，但最坏退化为 O(n^2)

def bubble_sort(lst):
    n = len(lst)
    for i in range(n - 1):
        for j in range(n - 1 - i):
            if lst[j] > lst[j + 1]:
                lst[j], lst[j + 1] = lst[j + 1], lst[j]
    return lst

def binary_search(lst, target):
    left, right = 0, len(lst) - 1
    while left <= right:
        mid = (left + right) // 2
        if lst[mid] == target:
            return mid
        elif lst[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1  # 未找到

二叉树公式
n0 = n2 + 1（叶子结点数 = 度为2的结点数 + 1）
第 k 层最多 2^(k-1) 个结点
深度为 k 的二叉树最多 2^k - 1 个结点
完全二叉树深度 = [log2(n)] + 1

推导关系

  • 设 n 为总结点数，n0 为叶子结点（度0），n1 为度1结点，n2 为度2结点
  • n = n0 + n1 + n2（总结点 = 各度结点之和）
  • 边数 = n - 1 = 0*n0 + 1*n1 + 2*n2
  • 联立可得：n0 = n2 + 1

满二叉树

  • 深度为 k 的满二叉树有 2^k - 1 个结点
  • 第 k 层有 2^(k-1) 个结点（全部为叶子）
树的分支数计算
分支数（边数） = 总结点数 - 1
分支数 = n1 + 2*n2 + 3*n3 + ... + k*nk
  • 度为 m 的结点有 m 个分支（出边）
  • 对于二叉树：分支数 = n1 + 2*n2
  • 任何树的分支数 = 总节点数 - 1（因为根节点没有入边）

class MyClass:
    """类的文档字符串"""

# 连接数据库（不存在则创建）
conn = sqlite3.connect("test.db")
cursor = conn.cursor()

# 创建表
cursor.execute("""
    CREATE TABLE IF NOT EXISTS students (
        id INTEGER PRIMARY KEY,
        name TEXT NOT NULL,
        age INTEGER,
        score REAL
    )
""")

# 插入数据
cursor.execute("INSERT INTO students VALUES (1, '张三', 20, 95.5)")
conn.commit()

# 查询数据
cursor.execute("SELECT * FROM students")
rows = cursor.fetchall()
for row in rows:
    print(row)

# 关闭连接
conn.close()
