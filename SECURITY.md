# 安全策略

## 项目说明

本项目是一个纯前端的 Python 交互式学习平台，所有代码运行在用户浏览器本地：

- 不涉及后端服务器
- 不收集用户个人信息
- 学习进度仅保存在浏览器 `localStorage` 中
- 所有数据内嵌在单个 HTML 文件中

## 报告安全问题

如果你发现了任何安全相关的问题（如 XSS 漏洞、代码注入等），请通过以下方式报告：

1. 在仓库中创建一个 **Issue**（标题前缀加上 `[Security]`）
2. 或通过 GitHub 私信仓库维护者

## 已知限制

- 本项目仅供学习和教育用途
- 不建议在生产环境中使用
- 所有 JavaScript 代码在客户端执行，请确保从可信来源访问

本项目按 "原样" 提供，不提供任何明示或暗示的担保。使用者需自行承担风险。
