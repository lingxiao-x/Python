# 社区行为准则

## 我们的承诺

作为本项目的维护者和贡献者，我们承诺为所有人提供一个开放、友好、包容的学习和交流环境，无论其年龄、体型、残疾、种族、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 虚心接受建设性的批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括：

- 使用带有性暗示的语言或图像，以及不受欢迎的性关注或性挑逗
- 发表挑衅、侮辱、贬低性评论，以及人身攻击或政治攻击
- 公开或私下骚扰他人
- 未经明确许可发布他人的私人信息（如住址、电子邮箱等）
- 其他在专业环境中被认为不恰当的行为

## 我们的责任

项目维护者有责任明确行为准则的标准，并对任何不可接受的行为采取适当和公正的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、问题报告和其他贡献，或暂时或永久地禁止任何他们认为有不恰当、威胁、冒犯或有害行为的贡献者。

## 适用范围

本行为准则适用于本项目所有的公共空间，也适用于个人在公共空间代表本项目或其社区时的行为。代表本项目的情况包括使用官方项目电子邮箱、通过官方社交媒体账号发帖，或在线上、线下活动中担任指定代表。

## 执行

如果你遭遇或目击了辱骂、骚扰或其他不可接受的行为，请通过 GitHub Issue 联系项目维护者。所有投诉都将被及时、公正地审查和调查。

项目维护者有义务对举报事件的相关人保密。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 1.4 版本。
